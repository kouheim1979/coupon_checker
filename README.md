# クーポンチェッカー

支払いチェッカーアプリの姉妹アプリとして、来店前に確認すべきクーポン・会員優待・決済優待の入口を検索する軽量Webアプリです。

## ファイル構成

- `index.html` : 単一HTML版の検索アプリ
- `coupon_database.json` : Webアプリ用DB
- `coupon_database.xlsx` : 更新・運用用Excel DB
- `README.md` : 使い方

## 目的

このアプリは、日替わりクーポン金額の完全収集ではなく、JAF、Club Off、YFC、Yahoo!クーポン、SmartNews、カード優待、店舗公式アプリ、ポイントアプリなどの「確認入口」を取りこぼさないことを目的にしています。

2. 使い方
1) coupon_checker.htmlをブラウザで開く。
2) 店名、カテゴリ、優待元で検索する。
3) 結果に表示された「確認する順番」に沿って、店舗公式アプリ、JAF、Club Off、SmartNews、Yahoo!、決済クーポン、カード優待を確認する。
4) 最後に支払いチェッカーアプリで決済方法を確認する。

3. JSON読込
- coupon_checker.htmlには初期DBを内蔵しているため、単体で動きます。
- Excelを更新してJSONを書き出した場合は、画面右側の「JSON DB読込」からcoupon_database.jsonを選択してください。

4. DB更新方法
- Excelの「定番クーポンDB」を編集します。
- 必須列:
  id / 店舗名 / 店舗名かな / ブランド名 / カテゴリ大 / カテゴリ小 / 優待元 / 優待元種別 / 優待ステータス / 優先度 / 信頼度 / 確認方法 / 確認URL / アプリ名 / 提示タイミング / 使う前の操作 / 支払いとの関係 / 併用注意 / 会員条件 / 対象範囲 / 店舗限定有無 / オンライン限定有無 / 最終確認日 / 次回確認目安 / メモ / 検索キーワード
- 公式情報がある場合は確認URLを入れます。
- 確認できない場合は空欄にせず「未確認」と入力します。
- 併用可否が不明な場合は「併用条件要確認」と入力します。
- 金額・割引率は変動しやすいため、主項目には入れません。

5. 優待ステータス
- あり:
  公式ページ、優待ページ、会員優待サイトなどで確認できる。
- 可能性あり:
  検索結果や過去情報では存在するが、現在有効か未確認。
- 要手動確認:
  アプリ内、ログイン後、紙券、会員限定、抽選、郵送など。
- なし:
  現時点では確認先が見当たらない。

6. 優先度
- 1. まず見る:
  店に着いてから最初に見る。
- 2. 次に見る:
  公式の次、またはカテゴリ的に強い優待元。
- 3. 余裕があれば見る:
  時間があれば確認。
- 4. 特定条件だけ見る:
  その支払い方法・カード・会員条件がある場合だけ見る。

7. 手動確認手順
1) 店舗公式アプリ/公式サイトを確認。
2) JAFで「{店舗名} JAF 優待」を確認。
3) Club Off/福利厚生で店名検索。
4) SmartNews、Yahoo! JAPANクーポンを確認。
5) PayPay、楽天ペイ、d払い、au PAY、Vpass/Vクーポンを確認。
6) エポス、イオンカード、JCB、Visa、Mastercard、アメックス優待を確認。
7) 確認ログに、確認日・結果・URL・メモ・次回確認目安を記録。

9. 注意
- このDBは「最新割引額の完全収集」ではなく、「確認先の取りこぼし防止」が目的です。
- 現在有効か不明なものを「あり」と断定しないでください。
- ログインが必要な会員サイト、アプリ内限定、スクレイピング禁止サイト、ワンタイムクーポン、個人別クーポン、紙券、郵送、抽選は自動取得せず「要手動確認」としてください。
